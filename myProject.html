My projects

Regular-studios fan-made repolished website

A self-initiated fan project inspired by the upcoming indie game by Regular Studios. I created a full website redesign to practice front-end development skills, including responsive layout, scroll animations, and interactive elements using HTML, CSS, and JavaScript.
This project is not affiliated with Regular Studio and was created purely for educational and creative purposes.

To-do list

A lightweight to-do list app developed as a beginner project to strengthen core front-end skills. Features include task creation, deletion, basic UI feedback and local storage to not lose previous to-do lists. Built with HTML, CSS, and JavaScript without any frameworks, focusing on DOM manipulation and user interaction.

Cyborg part-selling website

A concept landing page for a fictional tech company specializing in cybernetic body parts. This project was created to explore modern web design techniques, including responsive layouts, call-to-action sections, and product-style cards. Built with HTML, CSS, and JavaScript, with a focus on visual storytelling and sci-fi branding.
